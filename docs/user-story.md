## User story US01 - Criar tarefa

|               |                                                                |
| ------------- | :------------------------------------------------------------- |
| **Descrição** | O usuário vai ser capaz de cadastrar tarefas e gerencia-las, como editar, excluir e listar. A tarefa vai ter título e descrição, e também status de andamento. |
